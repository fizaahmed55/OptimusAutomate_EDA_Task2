# Optimus Automate Data Analytics Internship - Task 2

## Sales Performance Dashboard

This project was completed as part of the Data Analytics Virtual Internship at Optimus Automate.

## Project Overview

The goal of this project is to analyze sales data and build an interactive dashboard that presents key business performance insights. The dashboard helps users understand revenue trends, product performance, regional sales, and overall business growth.

## Features

- Total revenue analysis
- Sales and profit performance tracking
- Top-performing products
- Regional performance comparison
- Category-wise sales analysis
- Monthly and yearly trend analysis
- Interactive visual dashboard
- Business insights and recommendations

## Key Performance Indicators

- Total Revenue
- Total Sales
- Total Profit
- Top Products
- Regional Performance
- Category Performance
- Sales Trends

## Technologies Used

- HTML
- CSS
- JavaScript
- Data Analytics
- Data Visualization
- Dashboard Design

## How to Run the Project

1. Download or clone this repository.
2. Open the project folder.
3. Open the `index.html` file in your browser.
4. Explore the interactive sales dashboard.
